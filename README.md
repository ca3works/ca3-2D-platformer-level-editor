# ca3 2D platformer level editor
 2D platformer level editor

This is a basic 2D editor to create a tilemap editor for events and enemies, items and icons of 2D platformers.
It exports to a basic json.

Made with raylib, gemini and python, compiled with pyinstaller.
Windows build not heavily tested, mostly tested on ubuntu.

Please use the python script if there's any issue with the binaries.

Press F5 to save the level in the /levels/ subfolder, and F6 to load the saved file.

Add tiles icons to subfolders inside /assets/ as 32x32 pngs. 
Replace background.png inside /assets/ with your current level reference.png
